# MSSAkit_TB
Code fork from MSSAkit [https://github.com/anitasaraswati](https://github.com/anitasaraswati/MSSAkit)
Code has been adapted to suit iLOVECLIM model output, pre- and post-processing, and a dataclass function has been implemented to streamline a large amount of MSSA runs.
